# 🏥 Sandipani Gurukul – MBBS Abroad Consultancy Platform

**Sandipani Gurukul** is a modern, production-ready web platform built to guide aspiring medical students through the process of pursuing **MBBS abroad**.

Designed for real-world consultancy use, this platform bridges the gap between **students and globally recognized medical universities**, offering structured guidance, verified data, and end-to-end admission support.

---

## 🌐 Live Website

🚀 https://sandipanigurukulmbbs.com/

---

## 🎯 Purpose

This platform is built to:

* Help students explore **international MBBS opportunities**
* Provide **accurate, verified university data**
* Simplify the **admission & consultation process**
* Represent Sandipani Gurukul as a **professional digital consultancy brand**

## ⚙️ Getting Started

### 🔧 Prerequisites

* Node.js (LTS recommended)
* npm or yarn

---

### 📥 Installation

```bash
git clone https://github.com/Deepanshu-Surve/Sandipani-Gurukul-MBBS.git
cd Sandipani-Gurukul-MBBS
npm install
```

---

### ▶️ Run Development Server

```bash
npm run dev
```

👉 App runs at: `http://localhost:3000`

---

### 📦 Build for Production

```bash
npm run build
```

Output will be generated in the `dist/` folder.
